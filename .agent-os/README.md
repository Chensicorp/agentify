# Agentify

This directory is the **vendor-neutral operating layer** for this repository. It is the source of truth for how agents and humans work here, regardless of which AI platform or model is in use.

---

## Studio status

| Field | Value |
|---|---|
| Studio stage | **pre-activation** — run `activate-studio` before any task work |

---

## Start here

1. Read this file.
2. Check `state/current-focus.md`. If it reads `pre-activation`, run `workflows/activate-studio.md` before anything else.
3. Read `logs/handoffs/current-handoff.md` for the latest work state.
4. Read `project/boundaries.md` before touching any files.
5. Check `verification/commands.md` before running any command.

## Workflow contract

For any non-trivial task, follow this sequence:

1. **Check state** — read `state/current-focus.md` and `logs/handoffs/current-handoff.md`
2. **Select tier** — read `workflows/orchestrate.md`
3. **Investigate** — read `workflows/investigate.md`
4. **Plan** — read `workflows/plan.md`
5. **Implement** — read `workflows/implement.md`
6. **Verify** — read `workflows/verify.md`
7. **Review** — read `workflows/review.md`
8. **Handoff** — read `workflows/handoff.md`
